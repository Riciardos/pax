pax.units module¶

This sets up variables for the various unit abbreviations, ensuring we always have a ‘consistent’ unit system. You can change base_units without breaking the consistency of any physical relations

Please change comments in this code if you change any units. Better yet, don’t change any units!

WARNING
The unit system is not fully consistent, since the unit for energy is set separately from mass, time, and distance If you do mechanics this is a SERIOUSLY BAD IDEA We, however, only ever need mass for density
